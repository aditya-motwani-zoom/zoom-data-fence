The profiles file is used to configure your connection to the database.

Structure

The profiles file has the following top-level structure:

{% highlight yaml %} default-profile: dev profiles: dev: provider-name: SNOWFLAKE connection: snowflake: connection-string: jdbc:snowflake://your-dev-account.snowflakecomputing.com:443 {% endhighlight %}
